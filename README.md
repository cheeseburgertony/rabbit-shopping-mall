# rabbit-shopping-mall 
# 小兔鲜电商项目

## 一个基于Vue3的电商项目

## 项目简介
这是一个使用Vue.js构建的电商项目，旨在提供一个完整的电商平台，包括用户登录、商品浏览、购物车管理、订单结算支付和订单管理等功能。

## 目录结构
├── public/ # 静态资源
│ ├── index.html # 入口HTML文件
├── src/ # 源码
│ ├── apis/ # API请求接口文件
│ ├── assets/ # 项目资源，如图片、样式等
│ ├── components/ # Vue组件
│ ├── composables/ # 可组合的功能模块
│ ├── directives/ # 自定义指令
│ ├── router/ # 路由配置
│ ├── store/ # 状态管理
│ ├── styles/ # 样式文件(自定义主题)
│ ├── utils/ # 工具函数
│ ├── views/ # 视图组件
│ ├── App.vue # 根组件
│ ├── main.js # 入口文件
├── .gitignore # Git忽略文件配置
├── package.json # 项目依赖和脚本
└── README.md # 项目简介

## 功能描述
- 用户登录
- 商品浏览
- 购物车管理
- 订单结算
- 支付管理
- 个人中心
- 订单管理

## 技术栈
- 前端框架：Vue.js
- 路由管理：Vue Router
- 状态管理：Pinia
- HTTP请求：Axios
- UI框架：Element UI
